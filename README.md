# Space Shooter

Projeto desenvolvido em C++ utilizando SFML.

## Objetivo

Aprender desenvolvimento de jogos e arquitetura de engine utilizando conceitos inspirados em engines profissionais.

## Tecnologias

- C++17
- SFML 3
- Makefile
- Git

## Funcionalidades

- Player
- Sistema de Tiros
- Asteroids
- Colisão
